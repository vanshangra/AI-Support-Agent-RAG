## Navigation Guide

### Finding a Feature

**Want to modify Chat UI?**
```
client/ai-agent/src/pages/Chat.jsx
```

**Want to change button styles?**
```
client/ai-agent/src/components/UI.jsx
→ Button component
```

**Want to add API endpoint?**
```
server/routes/[feature]Routes.js
→ server/controllers/[feature]Controller.js
→ server/models/[Model].js
```

**Want to improve RAG retrieval?**
```
rag-service/services/retrieval_service.py
or
rag-service/services/embedding_service.py
```

**Want to change design colors?**
```
client/ai-agent/tailwind.config.js
or
client/ai-agent/src/index.css
```

### Common Tasks

#### Adding a New Page
1. Create file: `src/pages/NewPage.jsx`
2. Add route: `src/App.jsx`
3. Import components from `components/`
4. Use `useToast()` for notifications

#### Adding a New Component
1. Create file: `src/components/NewComponent.jsx`
2. Import from `./UI.jsx` if using base components
3. Export for use in pages
4. Document in `COMPONENT_EXAMPLES.md`

#### Adding an API Endpoint
1. Create route: `routes/featureRoutes.js`
2. Create controller: `controllers/featureController.js`
3. Add model if needed: `models/Feature.js`
4. Register route in: `server.js`

#### Adding a Database Model
1. Create file: `models/NewModel.js`
2. Define Schema: `new mongoose.Schema({...})`
3. Export: `module.exports = mongoose.model(...)`
4. Use in controller: `import NewModel from '../models/NewModel'`
